# macOS Sierra & High Sierra for Xiaomi Air 12.5
Hackintosh your Xiaomi Air 12.5


## Features

* Working
  * CPU
  * GPU acceleration
  * Audio
  * SSD
  * USB (A & C)
  * HDMI
  * Keyboard
  * Touchpad
  * Webcam

* Non-working
   * WIFI
   * Bluetooth
